#The awesome stream adventure

Follow the [stream-adventure](https://github.com/substack/stream-adventure) installation instructions and then in a terminal run:

```
stream-adventure
```

The files are:

  1. program.js
  2. pipe.js
  3. input-out.js
  4. transform.js
  5. lines.js
  6. concat.js
  7. http-server.js
  8. http-client.js
  9. websockets.js
  10. html-strem.js
  11. duplexer.js
  12. duplexer-redux.js
  13. combiner.js
  14. crypt.js

Thanks @substack!
